# Template for website with supabase

Create a github repo and clone it using this template.

```
gh repo create em-astro/[NAME] --private --clone --template=effektivmarkedsforing/astro-template
```

Starting a local Astro project with this template.

```
npm create astro@latest -- --template effektivmarkedsforing/astro-template -y
```

- Atkinson hyperlegible font
- Custom colors; check out your-url/colors
- Image api route
